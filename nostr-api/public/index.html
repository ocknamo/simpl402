⚡ x402 over Lightning Network - Nostr API

Lightning Network による支払いが必要な REST API です。x402 プロトコルに準拠しています。

🔌 API エンドポイント

GET /nostr/secret-key

Lightning 支払い検証後に Nostr の秘密鍵を返します。

🔄 使い方

1. 初回リクエスト 402 Payment Required

curl -i http://localhost:8787/nostr/secret-key

レスポンスに Base64 エンコードされた Lightning invoice を含む PAYMENT-REQUIRED ヘッダーが返されます。

2. Invoice のデコード

echo "eyJ..." | base64 -d | jq

結果：

{
  "scheme": "lightning",
  "network": "bitcoin",
  "invoice": "lnbc1..."
}

3. Invoice の支払い

Lightning ウォレットで invoice を支払います（デフォルト: 100 sats）。

4. 支払い後のアクセス 200 OK

curl -i http://localhost:8787/nostr/secret-key \
  -H "PAYMENT-SIGNATURE: eyJ..."

支払い検証後、秘密鍵が返されます。

🔒 セキュリティ

  • 各 invoice は一度のみ使用可能（Cloudflare KV で管理）
  • 期限切れ invoice は自動的に拒否
  • coinos.io Lightning ノードで支払い検証

🛠️ 技術スタック

  • Cloudflare Workers（サーバーレス）
  • Lightning Network（coinos.io API）
  • Cloudflare KV（使用済み invoice 追跡）
  • TypeScript
